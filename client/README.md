# Ventas Garzón - Frontend

Aplicación React con TypeScript, Tailwind CSS y Framer Motion.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- `src/components/` - Componentes reutilizables
- `src/pages/` - Páginas principales
- `src/services/` - Servicios API
- `src/store/` - Estado global con Zustand
- `src/hooks/` - Custom hooks
- `src/utils/` - Utilidades y animaciones
- `src/types/` - Tipos TypeScript
